# DocuMapper 
Script de automatización desarrollado en Python para la gestión y mapeo de procesos documentales.

### Funcionalidad
Este script recorre directorios locales y genera automáticamente un índice organizado en formato Markdown, facilitando el control de versiones y la auditoría de archivos técnicos.

### Tecnologías
* **Python 3.14**
* **Librería os** (Manejo de sistemas de archivos)

### Instrucciones
1. Ejecutar `python main.py` en la carpeta deseada.
2. El archivo `Indice_Documentacion.md` se generará instantáneamente con el listado de archivos.
